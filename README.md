# Description\nImg-Annotation-Tool is a graphical image annotation tool which can label many kinds of object type (Pen, Eraser, Hollow Rectangle, Filled Rectangle, Hollow Circle, Filled Circle, Hollow Ellipse, Filled Ellipse, Rectangle ROI, Irregular Shape ROI) in images.
